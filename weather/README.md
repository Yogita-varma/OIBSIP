Project 3 :Weather Information System

The Weather Information System is a Python application that fetches real-time
weather information for a given city or ZIP code using the OpenWeatherMap API.

Note : Add your API_KEY 

## Features
- Fetches real-time weather data
- Supports Celsius and Fahrenheit units
- Displays temperature, humidity, and weather condition
- Handles invalid input gracefully
