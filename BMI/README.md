project 1: BMI Calculator

The BMI (Body Mass Index) Calculator is a Python-based application that calculates
the BMI of a user using height and weight inputs. Based on the calculated value,
it categorizes the user into different health conditions.

##  Features
- Takes user input for height and weight
- Calculates BMI using the standard formula
- Displays BMI value and health category
- Uses CSV data for reference

## Files
- `main.py` – Main Python program
- `data.csv` – Dataset used in the project

##  How to Run
1. Make sure Python is installed
2. Open terminal in this folder
3. Run:
   ```bash
   python main.py
